# Password Strength Checker

## Description
A Python tool that evaluates the strength of a password based on security criteria.

## Features
- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Rates passwords as Weak, Medium, or Strong
- Provides suggestions for improvement
- Allows multiple password checks

## How to Run
1. Open the terminal.
2. Run:
   python password_checker.py
3. Enter a password to analyze.

## Technology Used
- Python 3
